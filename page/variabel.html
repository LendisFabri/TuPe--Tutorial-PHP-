Variabel

Definisi Variabel

Variabel merupakan sebuah tempat penyimpanan data yang berupa nilai. Variabel juga dapat diibaratkan seperti sebuah wadah penyimpanan.


Deklarasi Variabel

Dalam bahasa pemrograman PHP, variabel dimulai dengan tanda dollar ($), diikuti dengan nama variabel, dan diakhiri dengan tanda titik koma (;).

$string = "Hello TuPe";
$angka = 5;
$desimal = 27.5;

Setelah pernyataan di atas dieksekusi, kita bisa melihat bahwa variabel $string akan menyimpan nilai Hello TuPe, variabel $angka akan menyimpan nilai 5, dan variabel $desimal akan menyimpan nilai 27.5.

Sebuah variabel dapat memiliki sebuah nama pendek seperti ($a dan $b) atau nama-nama sesuai nilai yang akan dideskripsikan seperti ($string, $angka, $hobi, $total).

Selain itu variabel sendiri memiliki aturan dalam penggunaan namanya, antara lain:

  • # Sebuah variabel dimulai dengan menggunakan tanda dollar ($), lalu diikuti dengan nama variabel
  • # Sebuah nama variabel harus dimulai dengan sebuah kata atau karakter garis bawah (_)
  • # Sebuah nama variabel tidak dapat dimulai dengan sebuah angka
  • # Sebuah nama variabel hanya dapat mengandung karakter huruf-angka dan garis bawah (A-z, 0-9, dan _)

*Catatan: Perlu diingat bahwa nama variabel sangatlah sensitive.

PHP Merupakan Bahasa yang Memiliki Kemudahan

Dalam contoh di atas, perhatikan bahwa kita tidak harus memberitahukan tipe data pada variabel tersebut. PHP telah dengan otomatis mengkonversi variabel dengan tipe data yang benar, tergantung pada nilai yang benar. Sedangkan dalam bahasa pemrograman lain seperti C, C++, Python, dan Java. Seorang programmer diharuskan untuk mendeklarasikan nama dan tipe data sebuah variabel sebelum digunakan.

Cakupan Variabel Dalam PHP

Dalam PHP, variabel dapat dideklarasikan pada berbagai tempat dalam script. Cakupan dalam sebuah variabel merupakan bagian script, yang dimana variabel tersebut dapat digunakan sebagai referensi. Variabel memiliki 3 cakupan variabel yang berbeda, antara lain:

  • # Lokal
  • # Global
  • # Statis

Cakupan Global dan Lokal

Sebuah variabel dideklarasikan diluar sebuah fungsi yang memiliki sebuah cakupan global dan hanya dapat diakses diluar sebuah fungsi.

$string = "Hello TuPe"; //cakupan global
function diCoba(){
	echo	$string; 
	// menggunakan variabel $string dalam
	fungsi akan menimbulkan pesan error
}
diCoba();

echo $string;

Sebuah variabel dideklarasikan bersama sebuah fungsi yang memiliki cakupan lokal dan hanya dapat diakses bersama fungsi itu sendiri.

function diCoba(){
	$string = "Hello TuPe"; //cakupan lokal
	echo	$string; 
}

diCoba();
echo $string;
// menggunakan variabel $string dalam
fungsi akan menimbulkan pesan error;

Kata Kunci Global dan Statis

Kata kunci global digunakan untuk mengakses sebuah variabel global bersama dalam sebuah fungsi. Untuk melakukannya, gunakan kata kunci global sebelum variabel (dalam fungsi).

x = 5;
y = 5;
function diCoba(){
	global $x, $y; 
	$y = $x + $y; 
}

diCoba();
echo $y; // hasil = 10;

Normalnya, ketika sebuah fungsi telah selesai dieksekusi, semua variabel akan dihapus. Bagaimanapun, di waktu yang sama kita ingin sebuah lokal variabel tidak ingin dihapus. Kita butuh ini untuk pada kasus yang lain. Untuk melakukannya, kita gunakan kata kunci static ketika kita pertama kali mendeklarasikan sebuah variabel.

function diCoba(){
	static $x = 5; 
	echo $x; 
 $x++; 
}

diCoba();
diCoba();
diCoba();

Kemudian, setiap kali fungsi dipanggil, variabel masih memiliki informasi yang terkandung dari terakhir kali fungsi dipanggil.

*Catatan: Variabel ini masih bersifat lokal dalam fungsi.
